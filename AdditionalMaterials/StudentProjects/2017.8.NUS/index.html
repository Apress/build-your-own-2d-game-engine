新加坡国立大学 2017 暑期短期课程
Introduction to 2D Game Engine Development.
Final Games

These are the final games we build at the end of an intense 13-day short course at the School of Computing, National University of Singapore. In this course, we learned about Game Egnine API structure and development during the first five days, followed by two days of game development fundamental, and spent the last five days building games of students' choices.

All students in class are from China, some just completed their second year of college, others, after their third year. Some students have prior game development experience, while most did not. Based on the game engine described in this book, these games are built from scratch over five days, from conceptualization to prototyping, to playtesting, to these final results.

NOTE: You should run the games using the Chrome web browser. Explore 11 does not support AudioContext and Mozilla Firefox (v 39.0) does not support GLSL shaders and thus will not allow the games to run. Some of the games takes a little while to load (~10's of seconds), please be patient.
