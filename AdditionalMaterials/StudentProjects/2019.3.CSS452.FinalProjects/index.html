Winter 2019 CSS452: 2D Game Engine Development. Final Games

These are the final games we build during the last three weeks of class CSS452 in Winter Quarter 2019 as ways to demonstrate the features of the GTCS game engine we have learned to build in this class.

NOTE: You should run the games using the Chrome web browser. Explore 11 does not support AudioContext and Mozilla Firefox (v 39.0) does not support GLSL shaders and thus will not allow the games to run.

Balloon Valhalla

A tale based on the oft-forgotten Balloon-Norse scriptures, about two valiant heroes from the Balloon wars attempting to make their way into Balloon Valhalla. These two friends will never leave each other behind, so you must help both of them navigate through treacherous mazes so they can die a heroic death in the upper atmosphere. If either of them perishes on the way, the other will choose to not continue.

Naturally, as balloons, they always float up - so you must rotate the maze itself to help them make it through, all the while avoiding deadly spikes and collecting the keys that will open the many heavenly gates to valhalla. Will you help them achieve a heroic death, or will they merely perish?
